# MoneyLens

A full-stack financial management application built with React and Node.js.

## Project Structure

```
MoneyLens/
├── frontend/          # React frontend application
│   ├── app/          # Application source code
│   ├── public/       # Static assets
│   └── package.json
└── backend/          # Node.js backend server
    ├── controllers/  # Request handlers
    ├── models/       # Data models
    ├── routes/       # API routes
    └── server.ts     # Server entry point
```

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:

```bash
git clone https://github.com/ubaidjs/MoneyLens.git
cd MoneyLens
```

2. Install backend dependencies:

```bash
cd backend
npm install
```

3. Install frontend dependencies:

```bash
cd ../frontend
npm install
```

### Running the Application

#### Backend

```bash
cd backend
npm start
```

#### Frontend

```bash
cd frontend
npm run dev
```

## Technology Stack

### Frontend

- React
- TypeScript
- Vite
- React Router

### Backend

- Node.js
- TypeScript
- Express (assumed)
